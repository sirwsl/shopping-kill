### 一、resources目录说明（重点）
#### 1、html文件夹重点说明
html里面是前端资源文件，别在项目中运行，需要单独踢出来，用Tomcat或者Nginx运行。
直接在项目中运行是运行不出结果的，这个项目在打包时候是去除前端文件的，我只是为了方便，所以丢在一起了。
你可以直接使用前端的一些编译器运行前端文件，切记别直接点开就行。
